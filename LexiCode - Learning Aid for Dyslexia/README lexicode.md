# Lexicode – Python Learning Aid

## Overview
Lexicode helps learners practice **confusable letters** (`b*`, `d**`, `p*`, `q**`) using **syllable splitting** and **TTS**.  
It’s designed for **visual and auditory learning**, ideal for beginners.

## Features
- **Confusable Letters Highlighted:** `b*`, `d**`, `p*`, `q**` marked for easy recognition.  
- **Syllable Splitting:** Words broken into syllables using hyphens for clearer pronunciation.  
- **Triple-space Word Separation:** Helps visually distinguish words.  
- **Multi-line Input Support:** Process multiple lines of text at once.  
- **Text-to-Speech (TTS):**  
  - **Normal Mode:** Reads the entire processed text.  
  - **Letter-by-Letter Mode:** Pronounces each letter individually for precision.  
- **Offline TTS:** Works without internet using `pyttsx3`.

## Usage
1. Run `lexicode.py`  
2. Enter text (finish input with an empty line)  
3. Enable TTS (`y/n`) and choose mode (`1` or `2`)
